# Test Files for Upload Features

This directory contains test files for testing various upload features in the admin dashboard.

## Available Test Files

### 1. Monthly Assessment Test File
- **File**: `monthly-assessment-test.txt`
- **Purpose**: Test file for Monthly Assessments upload feature
- **Format**: TXT (Plain Text)
- **Usage**: 
  - Go to "Monthly Assessments" page
  - Click "Add Assessment"
  - Select this file in the "Report File" field
  - Fill in other required fields and upload

### 2. Psychological Evaluation Test File
- **File**: `psychological-evaluation-test.txt`
- **Purpose**: Test file for Psychological Evaluations upload feature
- **Format**: TXT (Plain Text)
- **Usage**:
  - Go to "Psychological Evaluations" page
  - Click "Add Evaluation"
  - Select this file in the "Report File" field
  - Fill in other required fields and upload

## Supported File Formats

The upload features support the following file formats:
- **Documents**: PDF, DOC, DOCX, TXT
- **Images**: JPG, JPEG, PNG

## Testing Different Formats

For testing PDF, DOC, DOCX, or image formats, you can:
1. Create a simple document in your preferred format
2. Use any existing document from your computer
3. Convert the TXT test files to other formats if needed

## Notes

- All test files are plain text files for easy editing
- You can modify the content of these files to test different scenarios
- File size should be reasonable (typically under 10MB)
- Make sure files don't contain sensitive information when testing
